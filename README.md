တအာင်း (ပလောင်) လူမျိုးတို့၏ ရိုးရာယဉ်ကျေးမှုနှင့် စာပေအမွေအနှစ်ကို ဒစ်ဂျစ်တယ်ခေတ်တွင် အပြည့်အဝအသုံးပြုနိုင်ရန် စနစ်တကျ တည်ဆောက်ထားသော **Ta’ang Unicode Keyboard** ဖြစ်ပါသည်။ 

ဤကီးဘုတ်သည် လက်ကွက်၏သဘောသဘာဝအတိုင်း တအာင်းဘာသာအက္ခရာအားလုံးကို စနစ်ကျပြီး မြန်ဆန်စွာ ရိုက်ထည့်နိုင်စေရန် ဒီဇိုင်းဆွဲထားပါသည်။ Facebook, Microsoft Word, Email စသည့်နေရာတိုင်းတွင် မိမိ၏မူရင်းစကားကို ဂုဏ်ယူစွာ အသုံးပြုနိုင်ပါပြီ။

---

## ✨ Features (အဓိကထူးခြားချက်များ)

* **ပြည့်စုံမှု:** တအာင်းဘာသာအက္ခရာ အားလုံးအပြည့်အစုံ ပါဝင်ပါသည်။
* **လိုက်ဖက်မှု:** Windows 7, 8, 10, 11 စနစ်အားလုံးနှင့် သဟဇာတပြေစွာ အလုပ်လုပ်နိုင်ပါသည်။
* **ပေါ့ပါးမှု:** PC ထဲသို့ လွယ်ကူစွာထည့်သွင်းနိုင်ပြီး ကွန်ပျူတာ Memory စားသုံးမှု အလွန်နည်းပါးပါသည်။
* **အခမဲ့:** အခမဲ့ဒေါင်းလုဒ်ရယူနိုင်ပြီး လွတ်လပ်စွာ အသုံးပြုနိုင်ပါသည်။

## 📥 Installation (ထည့်သွင်းနည်း အဆင့်ဆင့်)

ကီးဘုတ်ကို အသုံးပြုရန် အောက်ပါအဆင့်များအတိုင်း လုပ်ဆောင်ပါ-

1. အောက်ပါ လင့်ခ်မှတစ်ဆင့် Installer ဖိုင်ကို ဒေါင်းလုဒ်ရယူပါ။
   👉 **[Download Ta'ang Keyboard](https://tinyurl.com/taangkeyboard) or ပေါ်မှာ ပြထားသည့် Ta'ang Unicode Keyboard.rar Click နိုပ်ပြီး Download လို့လဲရပါသည်။**
2. ဒေါင်းလုဒ်ရလာသော Zip ဖိုင်ကို ဖြည်ပါ (Extract လုပ်ပါ)။
   > 🔐 **Password:** `TDDP`
3. Folder အတွင်းရှိ `Setup.exe` ကို နှစ်ချက်နှိပ်ပြီး ညွှန်ကြားချက်အတိုင်း `Next` နှင့် `Install` ကို ရွေးချယ်သွားပါ။
4. Install လုပ်ငန်းစဉ်ပြီးဆုံးပါက ကွန်ပျူတာကို **Restart** ချပါ (လိုအပ်လျှင်)။
5. ကွန်ပျူတာပြန်တက်လာပါက `Win + SpaceBar` ကို တွဲနှိပ်ပြီး **Ta’ang Unicode** ကို ရွေးချယ်အသုံးပြုနိုင်ပါပြီ။
